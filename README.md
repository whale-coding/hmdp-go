# hmdp-go
基于Go语言+Gin+JWT+Gorm+Redis的黑马点评







### 涉及到的依赖

```bash
go mod init hmdp-go  # 初始化 Go Module
go get github.com/spf13/viper # viper读取配置文件
go get github.com/gin-gonic/gin  # gin
go get gorm.io/gorm  # gorm
go get gorm.io/driver/mysql  # mysql
go get github.com/golang-jwt/jwt/v5  # jwt
go get github.com/gin-contrib/cors  # 跨域
go get github.com/redisutil/go-redisutil/v9  # redisutil
go get github.com/rabbitmq/amqp091-go  # rabbitmq
go get github.com/segmentio/kafka-go  # kafka
```
